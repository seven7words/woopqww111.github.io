关于git

在公司里跟git玩了很久很久很久，玩到自己崩溃了
使用的是sourcetree这个东西，以我用svn的经验来看，我觉得就是对于冲突我很没有把握，不敢处理，所以才会接二连三的出现了很多诡异的问题
尤其是竟然自己莫名其妙干出来了一个标签，一个临时分支。好吧，最后的解决方法是强行删掉，重新更改。然后把自己的程序跟服务器达到同一水平上。
由于svn用的比较多，所以认真找了一下，发现了sourcetree的冲突解决方案的一个小插件叫做beyondcompare。用它可以分析一下两者代码的差异进行具体的改变。
